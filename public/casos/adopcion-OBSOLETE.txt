adopcion.json is no longer read by the site build.

The source of truth for adoption dogs is now:
  src/content/adoption-dogs/*.md   (dog metadata + image references)
  src/assets/casos/adopcion/       (source images, optimized by Astro at build time)

This file continues to be regenerated by:
  npm run prepare:casos

That script also generates exito.json (still used at runtime by StoriesSection.astro),
so the script is preserved. The adoption feed section of the script is now obsolete.
