Decisiones clave

  • Charcoal sobre color, no blanco. El cyan no pasa AA con texto blanco. CTAs usan texto charcoal sobre naranja (9.1 : 1).
  • Naranja para CTAs, no cyan. El cyan es para enlaces y bordes. El warm orange es la única "voz de acción".
  • Rainbow set en pequeñas dosis. Sólo divisores, números de stepper, chips de tarjeta. Nunca fondos grandes ni body text.
  • Sin páginas de detalle por perro. La adopción es una sola lista. Cada tarjeta hace deep-link directo a WhatsApp.
  • Privacidad por diseño. Ningún layout expone nombres, fotos del equipo o direcciones físicas.

05 · Contrast & accessibility

Contraste.

Validación WCAG AA de las combinaciones del sistema. Las que fallan están explícitamente prohibidas.

CombinaciónRatioVeredicto
Charcoal sobre blanco15.4 : 1✅ AAA
Charcoal sobre --color-surface14.1 : 1✅ AAA
Charcoal sobre cyan8.2 : 1✅ AAA
Charcoal sobre naranja (CTA)9.1 : 1✅ AAA — preferido
Blanco sobre magenta4.9 : 1⚠️ AA solo texto grande (≥18px bold)
Blanco sobre cyan2.6 : 1❌ Falla AA — nunca
Blanco sobre naranja1.7 : 1❌ Falla todo — nunca

Reglas de oro

  • Cuando dudes, charcoal sobre color. Nunca blanco sobre cyan o naranja.
  • El donation banner (magenta) usa cuerpo de 18px+ y peso 500+ para satisfacer AA-large.
  • Todos los inputs y botones tienen altura mínima 48px (target a11y).
  • Focus state: anillo cyan de 3px (--shadow-focus).
